# Trabalho M1 - Lógica Proposicional e Estruturas de Condição

Projeto desenvolvido para a disciplina de Algoritmos e Programação.

## Sobre o projeto

O programa foi desenvolvido em linguagem C com o objetivo de aplicar conceitos de:

- Lógica proposicional
- Estruturas condicionais
- Operadores lógicos
- Fluxogramas
- Tabela verdade

## Tecnologias utilizadas

- Linguagem C
- GCC
- Visual Studio Code

## Estrutura

- `TrabalhoM1.c` → Código principal
- `fluxograma.png` → Fluxograma do sistema
